This section describes which type of game is being created.

If you choose between these types, you must obey these rules:

Student - Skip this section by pressing check 2.

PC () - Fill the "type" and "descriptor/question" (make sure every box is filled in ) and then press check 2.

KID () - Fill the "image" , "type" and "descriptor/question" (make sure every box is filled in ) and then press check 2.

If you are creating game for certain type (PC / kids / students ) make sure you do not mix the options.

Be sure to delete unused rows.

If you are creating question game type do not use characters as /\'"?| , the question will be formed as : Your image " is/has + part of your question (is a piano player / has a piano )" ?

If you decide to create a game for Kids, you automatically create a game for PC and Students too.

If you decide to create a game for PC, you automatically create a game for Students too.

If you decide to create a game for Students, the game will be for Students only.

You have an option to create a question game type by skipping to choose an image. This game will also be for all types.
